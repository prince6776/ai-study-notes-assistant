# Raw Data Ingestion (Bronze Layer)

## Sources Ingested:
- **Wikipedia**: AI, Machine Learning, NLP, Computer Vision articles
- **Class notes**: Lecture slides/text from data science course  
- **Study guides**: Online AI/ML intro resources

## How ingested:
1. Browsed Wikipedia → copied raw article text
2. Downloaded lecture PDFs → extracted text  
3. Collected study guide markdown

**Raw format**: Unstructured text/markdown files
**Next**: ETL → cleaned → processed-notes.json (Gold layer)

*POC used manual copy-paste for simplicity. Production = automated API/scraping.*
